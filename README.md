## 1. 批次提取影片 URL

**功能說明**

輸入 jw.org 影片分類頁面的網址，工具會自動呼叫 JW API，取得該分類下所有影片的連結清單，並支援動態語言轉換。

**使用步驟**

1. 開啟工具，在輸入框貼上 jw.org 影片分類頁網址
   - 範例：`https://www.jw.org/cmn-hant/多媒體圖書館/影片/#cmn-hant/categories/StudioMonthlyPrograms`
2. 點擊「**立即提取影片**」按鈕
3. 等待清單載入完成後，可進行以下操作：
   - **全選／取消全選**：快速選取所有連結
   - **區間選取**：輸入起訖編號後點「選取」，例如第 1 ～ 10 筆
   - **勾選個別項目**：手動勾選想要的影片
4. 點擊「**複製已選取**」將選取的連結複製到剪貼簿

**注意事項**

- 網址必須來自 jw.org 的影片分類頁（含 `categories/` 路徑段落）
- 工具會自動從網址中偵測語言代碼（如 `cmn-hant`、`E` 等）並轉換

## 2. 提取 jw.org 影片字幕

**功能說明**

輸入一個或多個 jw.org 影片連結，工具會自動抓取各影片的 VTT 字幕檔並整理成可閱讀格式，支援下載為 PDF、HTML 或 Markdown。

**使用步驟**

1. 開啟工具，在輸入區貼上影片網址
   - 批次處理：每行貼一個網址（換行分隔），**一次最多 200 部**
2. 點擊「**提取**」按鈕，等待字幕載入
3. 提取完成後，使用右下角的操作按鈕：
   - 🔴 **PDF**：列印或儲存為 PDF 格式
   - 🟣 **HTML**：下載網頁版（保留排版）
   - 🔵 **MD**：下載 Markdown 版（純文字，方便後續編輯）
   - ⬛ **複製**：將 Markdown 內容複製到剪貼簿

**注意事項**

- 部分影片若無字幕檔，提取時會顯示「此影片無字幕檔」
- 可在下載前修改輸出檔名

## 3. 抓取線上書庫文章

**功能說明**

輸入守望台線上書庫（WOL）的目錄頁網址，工具會自動展開目錄結構（最多兩層子目錄），列出所有文章連結，選取後批次提取內容，並下載為 MD 或 HTML 格式。

**使用步驟**

1. 開啟工具，在「**目標目錄網址**」欄位貼上 WOL 目錄頁網址
   - 範例：某出版物的索引頁連結
2. 點擊「**1. 獲取頁面連結**」，工具會自動爬取目錄並列出所有文章，預設全部勾選
3. 選擇想提取的文章，支援三種方式：
   - **全選／取消全選**：一鍵切換
   - **編號範圍**：輸入起訖編號後點「勾選此範圍」
   - **關鍵字篩選**：輸入關鍵字後點「勾選篩選結果」
4. 確認勾選後點擊「**2. 開始提取內容**」，等待完成
5. 提取完成後選擇下載格式：
   - **下載 MD**：Markdown 格式，保留連結與純文字
   - **下載 HTML（推薦）**：保留原始排版，閱讀體驗最佳

**注意事項**

- 提取速度會刻意放慢（每頁間隔約 1 秒），以避免被網站封鎖，文章數量多時請耐心等候
- 相對連結會自動轉換為絕對連結，確保離線閱讀可正常連結

## 4. HTML 合併工具

**功能說明**

將多個 HTML 檔案合併為單一 HTML 檔案，自動整合 CSS 樣式，方便分享或封存。完全在瀏覽器本地執行，不上傳任何資料。

**使用步驟**

1. 開啟工具，將 `.html` 或 `.htm` 檔案拖曳至上傳區，或點擊上傳區選取檔案
2. 檔案載入後，可調整合併順序：
   - 點擊 ↑ ↓ 按鈕調整各檔案的前後順序
   - 點擊 🗑 按鈕移除不需要的檔案
   - 點擊「**加入更多**」繼續新增檔案
3. 點擊「**開始合併**」
4. 在檔名欄位輸入想要的輸出檔名，點擊「**下載**」即可

**注意事項**

- 合併時會自動提取各檔案 `<head>` 中的 `<style>` 與 `<link>` 樣式，並去除重複
- 每個檔案的內容會以 `<div class="merged-section">` 包裹，並加上來源檔名標注

## 5. Markdown 合併工具

**功能說明**

將多個 `.md` 或 `.markdown` 檔案合併為單一 Markdown 檔案，支援拖曳排序，方便整理筆記或批次匯出的文件。

**使用步驟**

1. 開啟工具，將 `.md` 檔案拖曳至上傳區，或點擊上傳區選取檔案（可一次選取多個）
2. 上傳後，可拖曳各列調整合併順序，或點擊 ✕ 移除不需要的檔案
3. 在「**下載檔名**」欄位輸入想要的輸出檔名（無需輸入副檔名）
4. 點擊「**⬇ 合併並下載**」即可

**注意事項**

- 每個檔案之間會以兩個空行分隔
- 重複上傳相同檔名的檔案不會重複加入

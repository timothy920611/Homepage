研讀工具箱

集合了jw.org影片網址擷取、jw.org影片字幕處理、線上書庫文章擷取與 HTML 檔案合併的實用小工具。⚠️以下工具非組織官方⚠️

Take URL

從jw.org網站批次擷取影片網址工具。例如輸入 https://www.jw.org/cmn-hant/多媒體圖書館/影片/#cmn-hant/categories/StudioMonthlyPrograms可以提取所有每月節目的連結。

多影片字幕工具

提去jw.org影片的字幕。以換行分割連結，一次最多提取50部影片。提取後可選擇下載pdf,html,txt檔。

Take Article

守望台線上書庫文章擷取閱讀器。輸入書庫目錄頁面連結可以提取目錄下的所有文章，並下載txt, html檔。

HTML 合併工具

能快速將多個 HTML 網頁檔案合併成一個單一檔案，方便分享或封存使用。

Markdown 合併工具

能快速將多個 Markdown 檔案合併成一個單一檔案，方便整理筆記、文件或封存使用。
